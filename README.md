## 🧰 Getting Started


### 💻 Prerequisites


```
npm install
```

### 🏃 Run locally

Would like to run and test this project locally? Open terminal and follow these steps:

1. Clone the repo
    ```sh
    git clone https://github.com/Skaistev/cypress-form.git 
    ```
2. Install NPM packages
    ```sh
    npm i
    ```
    or
    ```sh
    npm install
    ```
3. Run the script 
   ```sh
    npm run start
   ```
